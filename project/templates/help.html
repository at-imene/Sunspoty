How to Use Sunspoty?

Wolf Number Calculation

  • Upload Full Disc Image:To calculate the Wolf number, start by uploading an image of the full disc of the sun. Ensure that the image contains visible sunspots.
  • Configure Parameters (Optional): If desired, you can configure the parameters for the calculation. Enter the desired values for the kernel size of the Sobel operator, DBSCAN epsilon, and sensitivity. If you prefer to use the default settings, you can skip this step.
  • Submit and Calculate:Once the image and parameters (if applicable) are set, click the "Submit" button to initiate the calculation process. Sunspoty will process the image and display the calculated Wolf number based on the detected sunspots.

Sunspot Classification

  • Upload Magnetic Image: For sunspot classification, upload a magnetic image of a sunspot. This image should highlight the magnetic properties of the sunspot.
  • Submit and Process:After uploading the magnetic image, click the "Submit" button to initiate the classification process. Sunspoty's classification model will automatically process the image and determine the sunspot's classification.
  • View Classification Result:Once the processing is complete, Sunspoty will display the classification result, indicating whether the sunspot is categorized as alpha, beta, or betax.
